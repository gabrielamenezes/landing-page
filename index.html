O celular Obabox é um celular para idoso?

O ObaSmart é sim um smartphone para idoso, mas não somente para o público mais experiente. Ele foi desenvolvido com as principais características de um celular moderno e desempenha as principais funções de um smartphone.

O celular Obasmart é um celular simples para idoso?

Sim! As configurações do Obasmart fazem dele um celular simples para idoso conectar-se com seus amigos e familiares.

O que faz com que o ObaSmart seja um celular simples para idoso?

O ObaSmart foi planejado pensado exatamente no público com maior dificuldade ou pouca familiaridade com a tecnologia. Ele já vem de fábrica com um sistema que deixa as letras e ícones da tela maiores, tornando o seu layout mais agradável e fácil de navegar.

É importante ressaltar que o aplicativo responsável por deixar o ObaSmart com um layout mais agradável e lúdico já está instalado, não é preciso baixar nenhum aplicativo.

O ObaSmart já vem com whatsapp?

Sim, o Obasmart é um celular para idosos com whatsapp, facebook, uber e outros aplicativos mais importantes e utilizados já instalados. Eles já estão prontos para usar, basta conectar a conta do usuário.
